## üì¶ Description

**YAIM904 d15b ‚Äì Minimum Electrical Consumption** reduces the **overall electrical energy consumption of all entities**.

This MOD is intended to make factories more energy-efficient by lowering power usage globally, helping players manage large-scale production setups with fewer power-generation requirements.

It affects both active and inactive entities, resulting in a generally lower electricity demand across the factory.

---

## ‚öôÔ∏è How it works (Player view)

* All entities consume **less electricity than in vanilla**
* Power reduction is applied automatically
* No configuration or player interaction is required
* Gameplay mechanics remain the same, only energy usage changes

The MOD works transparently once enabled.
